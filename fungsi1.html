PENGERTIAN WEB

Aplikasi berbasis web adalah aplikasi yang berjalan melalui browser seperti Chrome, Firefox, dan Edge tanpa perlu instalasi tambahan pada perangkat pengguna.

Cara kerjanya: browser mengirim permintaan (request) ke server, lalu server mengolah data dan mengirimkan hasilnya kembali ke browser dalam bentuk halaman web.

Teknologi utama web terdiri dari:
• HTML – struktur halaman
• CSS – desain/tampilan
• JavaScript – interaksi & fungsi

Untuk sisi server digunakan bahasa seperti:
• PHP
• Python
• Node.js
• Ruby
• Java

Manfaat Coding

1. Meningkatkan kemampuan problem solving.

2. Membuka peluang karir besar.

3. Mampu membuat aplikasi sendiri.

4. Mengembangkan kreativitas.

5. Membantu membuat otomasi/mempercepat pekerjaan.

Jenis Bahasa Pemrograman

1. Python — mudah dipelajari dan fleksibel.

2. JavaScript — bahasa utama untuk web interaktif.

3. Java — terkenal di aplikasi enterprise dan Android.

4. C++ — untuk performa tinggi seperti game/engine.

5. Ruby — sintaks sederhana dan mudah dibaca.

6. Swift — bahasa resmi Apple untuk iOS/macOS.

Cara Mempelajari Coding

1. Pilih bahasa sesuai tujuan kamu.

2. Gunakan sumber online seperti YouTube / docs.

3. Baca dokumentasi resmi bahasa tersebut.

4. Gabung komunitas programmer.

5. Mulai dari proyek kecil.

6. Ikuti bootcamp jika ingin fokus.

7. Latihan rutin + konsisten.
